# feath

## ■ サービス概要
- 愛鳥や野鳥の画像や動画を気軽に投稿しシェアできる、鳥を愛する人のためのSNSです。ユーザー同士で投稿に「いいね」や愛鳥のフォローをし合うことができます。

- 日常の投稿だけでなく、迷子になった鳥の情報や保護した鳥の情報、また里親の募集の投稿可能です。アプリ上のマップを通じて、近くで迷子になった鳥の情報や保護された鳥、里親を募集している鳥の情報がシェアされます。

- 迷子や保護の情報は近くに住むユーザーに通知されます。さらに、保護情報の投稿時には「写真」「見つけた場所」「鳥の種類」「色」などの情報をもとに、アプリ内のすべての画像や情報から飼い主の可能性が高い人を予測します。これにより、迷子になった鳥が無事にお家に帰るのをサポートします。

<br>

## ■ このサービスへの思い・作りたい理由
私は1年ほど前からセキセイインコを飼い始めました。その影響で鳥が好きになり、Xやインスタグラム、YouTubeでインコや文鳥などの画像や動画を見ることが増えました。愛らしい鳥たちの姿に癒される一方で、インターネット上で迷子になった愛鳥の情報を求める声や、迷い鳥を保護して飼い主を探している投稿にも触れるようになりました。

鳥好きのためのコミュニティかつ、鳥のために何か価値を提供できるサービスとして、鳥に特化したモダンなSNSを作りたいと思いました。このサービスはユーザーが気軽に愛らしい鳥の画像や動画をシェアし楽しみながら、同時に投稿された画像、登録されたユーザ・愛鳥の情報から飼い主の予測やコミュニティ全体で支援することで、迷子の鳥が早く家に帰れるようにサポートします。

このアプリケーションは、鳥を愛する人々が安心して楽しみながら、迷子の鳥や保護された鳥が一日でも早く飼い主の元に帰ることを目的としています。

<br>

## ■ ユーザー層について
### *① 現在鳥を飼っている人*
本サービスで愛鳥の画像や動画をシェアすることができます。また愛鳥が迷子になってしまった際や、その他の事態において本サービスでのサポートおよび他のユーザにサポートを呼びかけることができます。

### ② 動物の画像や動画を見るのが好きな人
本サービスで、愛らしい鳥の画像や動画を気軽に見ることができ、特定のアカウントもしくは愛鳥をフォローすることで、投稿の通知を受け取ったりパーソナライズされたタイムラインを閲覧することができます。

<br>

## ■サービスの利用イメージ
- モバイルファーストなUI/UXを意識したデザインやフロントエンドフレームワークを活用した開発により、PCのみでなくスマートフォンからもネイティブアプリケーションのような体験を提供します。

- X（Twitter）やInstagramなど主流なSNSと同じように、ユーザはスマホからいつでもアプリケーションをアクセスおよび利用することができ、撮影した鳥の画像をスピーディに投稿すること他ユーザの投稿を見て楽しむことができます。

- 場所を選ばずに利用がしやすいことから、迷子情報や保護情報、目撃情報などをすぐにアプリケーションでシェアすることができます。

- ユーザは自身および他ユーザの投稿をXやInstagramなど他SNSにシェアすることができます。

<br>

## ■ ユーザーの獲得について
- 公式Xアカウントや公式インスタグラムアカウントなどを作成し、アプリケーションのプロモーションを実施します。

- ユーザが自身、他ユーザの投稿をXやInstagramなど他SNSでのシェアをすることにより、本サービスへの導線を作ります。

- また、いいね数やインプレッションの多い投稿を公式Xで紹介することにより、ユーザのエンゲージメントの向上を促進しつつ、新規ユーザの発掘する機会を創出します。

<br>

## ■ サービスの差別化ポイント・推しポイント
### 類似サービス、競合サービスとの差別化について
鳥専用SNSを謳うサービスは調べる限り1件ありました。そのサービスは育て方から健康情報、病院リストや図鑑など鳥に関する情報量は豊富にある素晴らしいサービスである反面、情報量や提供するサービスが多いことから、自身が投稿を作成する・他ユーザの投稿を閲覧するまでの導線が長く、これらの利用をするまでのハードルがやや高く感じました。

本サービスは、アプリ側から提供する情報量を絞り、ユーザが投稿を作成および他ユーザの投稿の閲覧やいいね、フォローするまでの導線・ハードルを可能な限り最小化することでユーザのエンゲージメントの最大化を目指します。

### X（Twitter）やInstagramとの差別化について
X（Twitter）やInstagramは特定のユーザ層に限定しない幅広い層のユーザが利用していることから、本サービスから同サービスへのシェア機能などを実装することにより、ユーザが併用しやすいUI・UXを目指します。

上記に加え、フォローやいいねした投稿から、好みの鳥種別や色などを予測しパーソナライズされたタイムラインの表示や、迷子情報や保護情報を投稿し飼い主の特定をサポートする機能などを提供することにより差別化を計ります。

## ■ 機能の実装方針予定
- トップページ
- ユーザ登録, ログイン
    - ユーザは「メールアドレス」「パスワード」「ユーザ名」を入力し、ユーザ登録をすることができます。
    - ユーザ登録後、「メールアドレス」「パスワード」によりログインし、本サービスのすべての機能を利用できます。
    - 実装にはgem `devise` の利用を予定しています。
- ゲストログイン
    - 本サービスは上記のユーザ登録をせずに、ポスト一覧の閲覧など一部の機能を利用ができます。
- 愛鳥の登録
    - ユーザは愛鳥の登録およびプロフィールを作成することができます。
    - プロフィールには「アイコン」「名前」「生年月日」「出生地」「種目」「紹介文」が含まれます。
- プロフィールページ
    - 自身および任意のユーザの「プロフィール」「登録した愛鳥」「住んでいる都道府県」「投稿したポスト」「いいねしたポスト」を閲覧することができる
    - 自身のプロフィールページの場合の作成・編集ができる
- ポストの作成
    - ユーザはポストを作成することができます。
    - ポストは「鳥の画像(複数可)or鳥の動画」「本文」「愛鳥の関連付け」「タグ」などが含まれます。
    - ポストに含まれる画像や動画は `Google Cloud Storage` にて管理を予定しています。
    - ユーザは作成したポストの編集および削除ができます。
- タグの予測
    - ポストの作成時に選択された画像（動画）からタグを予測し作成します。
    - タグの予測には `Google Cloud Vision` の利用を予定しています。
- ポスト一覧の取得・閲覧
    - ユーザは自身および他ユーザの作成したポストを閲覧することができます。
- ポストへのコメント、いいね
    - ユーザは任意のポストへコメントを投稿することができます。
    - ユーザは任意のポストにいいねをすることができます。
- フォロー
    - ユーザは任意の子（登録された愛鳥）および任意のユーザをフォローすることができる。
    - ユーザはフォローした子およびユーザの投稿を、すべての投稿とは別で閲覧することができます

- 掲示板機能
    - ユーザは任意のカテゴリで掲示板を作成できる
    - カテゴリは「迷子情報」「保護情報」「里親募集」「健康・病院」「その他・雑談」
    - 「迷子情報」「保護情報」「里親募集」には位置情報（都道府県以外非公開）を設定できる
- マップ機能
    - 位置情報から半径10km以内の「迷子情報」「保護情報」「里親募集」の掲示板情報が表示される
    - 自身の住んでいる都道府県に住んでいる他ユーザの愛鳥が表示される
    - 自身の住んでいる都道府県で出征した他ユーザの愛鳥が表示される
- DM機能
    - 任意の他のユーザにプロフィールページからDMを送ることができる
    - やりとりの履歴はDMページに表示される
- お知らせ機能
    - フォローしたユーザ、他ユーザの愛鳥の新着投稿のお知らせ
    - 自身および自身の愛鳥への新規フォロワーのお知らせ
    - 自身が住んでいる住んでいる都道府県、現在の位置情報から新着の「迷子情報」「保護情報」「里親募集」の掲示板情報のお知らせ
    - 新着のDMのお知らせ
